Background-color

Определяет цвет фона элемента

Определяет цвет фона элемента. По умолчанию без цвета.(Прозрачный)

Определяет цвет фона элемента. С заданым цветом. (Зелений). CSS: {background: #150}

Определяет цвет фона элемента. С заданым цветом. (Зелений) CSS: {background: rgb(17, 85, 0)}

Определяет цвет фона элемента. С заданым цветом. (Зелений) CSS: {background: rgba(17, 85, 0, 0.5)}

Background-image

Устанавливает фоновое изображение для элемента либо задает градиентную заливку

Устанавливает фоновое изображение для элемента либо задает градиентную заливку. (Картинка) CSS:{background: url('Путь к картинке.')}

Устанавливает фоновое изображение для элемента либо задает градиентную заливку. (Линейный градиент) CSS:{background: linear-gradient(to bottom, rgba(255, 0, 0, 1) 0%,rgba(94,83,115,1) 50%,rgba(24,181,164,1) 100%)}

Устанавливает фоновое изображение для элемента либо задает градиентную заливку. (Задиальний градиент) CSS:{background: radial-gradient(ellipse at center, rgba(30,102,157,1) 0%, rgba(255, 0, 0, 1) 0%,rgba(24,181,164,1) 100%)}

Background-repeat

Определяет, как будет повторяться фоновое изображение.

По умолчанию. Фоновое изображение повторяется по горизонтали и вертикали. CSS:{background: url('Путь к картинке.')}

No-repeat. Устанавливает одно фоновое изображение в элементе без его повторений. CSS:{background: url('Путь к картинке.') no-repeat}

Repeat-x. Фоновый рисунок повторяется только по горизонтали. CSS:{background: url('Путь к картинке.') repeat-x}

Repeat-y. Фоновый рисунок повторяется только по вертикали. CSS:{background: url('Путь к картинке.') repeat-y}

Spase. Изображение повторяется столько раз, чтобы полностью заполнить область; если это не удаётся, между картинками добавляется пустое пространство. CSS:{background: url('Путь к картинке.') space}

Round. Изображение повторяется так, чтобы в области поместилось целое число рисунков; если это не удаётся сделать, то фоновые рисунки масштабируются. CSS:{background: url('Путь к картинке.') round}

Background-position

Задает начальное положение фонового изображения, где первое значение (проценты, px, слово) определяет расположение по горизонтали, второе - вертикали.

По умолчанию. CSS:{background: url('Путь к картинке.') 0 0 no-repeat}

По центру. CSS:{background: url('Путь к картинке.') 50% no-repeat или CSS:{background: url('Путь к картинке.')} center no-repeat

Слева. CSS:{background: url('Путь к картинке.') 0 50% no-repeat или CSS:{background: url('Путь к картинке.') left no-repeat}

В углу. CSS:{background: url('Путь к картинке.') 100% 100% no-repeat или CSS:{background: url('Путь к картинке.') bottom right no-repeat}

Произвольные параметры. CSS:{background: url('Путь к картинке.') 50px 25px no-repeat}

Background-attachment

Устанавливает, будет ли прокручиваться фоновое изображение вместе с содержимым элемента.

По умолчанию. Позволяет перемещаться фону вместе с окном браузера CSS:{background: url('Путь к картинке.') 0 0 no-repeat scroll}

Fixed. Делает фоновое изображение элемента неподвижным, а так же устанавливает фоновое изображение относительно окна браузера. CSS:{background: url('Путь к картинке.') 0 0 no-repeat fixed}

Local. Фон фиксируется с учётом поведения элемента. Если элемент имеет прокрутку, то фон будет прокручиваться вместе с содержимым, но фон выходящий за рамки элемента остаётся на месте. CSS:{background: url('Путь к картинке.') 0 0 no-repeat local

Background-size

Масштабирует фоновое изображение согласно заданным размерам. В цифрах, где первое значение (проценты, px) определяет ширину, второе - высоту. Либо специальным значением.

Размеры картинки по умолчанию. CSS:{background: url('Путь к картинке.') 0 0/auto auto no-repeat}

Размеры картинки по ширине блока. CSS:{background: url('Путь к картинке.') 0 0/100% auto no-repeat}

Размеры картинки по высоте блока. CSS:{background: url('Путь к картинке.') 0 0/auto 100% no-repeat}

Расстягиваем картинку на всю ширину и высоту блока. CSS:{background: url('Путь к картинке.') 0 0/100% 100% no-repeat}

Масштабирует изображение с сохранением пропорций таким образом, чтобы картинка целиком поместилась внутрь блока. То есть по меньшей стороне картинки. CSS:{background: url('Путь к картинке.') 0 0/contain no-repeat}

Масштабирует изображение с сохранением пропорций так, чтобы его ширина или высота равнялась ширине или высоте блока. То есть по большей стороне картинки. CSS:{background: url('Путь к картинке.') 0 0/cover no-repeat}

Устанавливаем конкретные размеры. Если установлено одно значение, оно задает ширину фона, второе значение принимается за auto. Пропорции картинки при этом сохраняются. Использование двух значений через пробел задает ширину и высоту фоновой картинки. CSS:{background: url('Путь к картинке.') 0 0/300px 400px no-repeat}

Несколько фоновых изображений

Свойство background позволяет задать несколько фоновых изображений одному блоку, причем со своими настройками позиционирования масштабирования и прокрутки. Указывать значения следует через запятую.

Цвет фона + одна картинка. CSS:{background:цвет фона url('Путь к картинке.')} no-repeat

Две картинки, у каждой свои настройки. Каждая следующая ложится под предыдущую. CSS:{background: url('Путь к картинке.') right no-repeat, url('Путь к картинке.') 50%/cover;}

Три картинки, у каждой свои настройки. Каждая следующая ложится под предыдущую. CSS:{background:url('Путь к картинке.') bottom left/70px no-repeat, url('Путь к картинке.') right no-repeat, url('Путь к картинке.') 50%/cover;}

Четыре картинки, у каждой свои настройки. Каждая следующая ложится под предыдущую. CSS:{background: url('Путь к картинке.') center no-repeat, url('Путь к картинке.') bottom left/40px no-repeat, url('.Путь к картинке.') top right/40px no-repeat, url('Путь к картинке.') 50%/cover no-repeat;;}
